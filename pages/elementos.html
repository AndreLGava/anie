Elementos

Elementos adicionais usando classes do ANIE e as existentes do Bootstrap.

Estrutura

Para comportar toda a estrutura do projeto o ANIE tem a classe .area, com tamanhos pré definidos conforme sua necessidade:

<div class="area-min">
  max-width: 980px;
</div>


<div class="area">
  max-width: 1180px;
</div>


<div class="area-max">
  max-width: 2000px;
</div>

Circulos

Classe para criar destaques de icones em circulos, use as classes .circle-min, .circle ou .circle-max conforme o tamanho que deseja, use também a classe da cor, exemplo:

<span class="circle-min red glyphicon glyphicon-heart" aria-hidden="true"></span>
<span class="circle red glyphicon glyphicon-heart" aria-hidden="true"></span>
<span class="circle-max red glyphicon glyphicon-heart" aria-hidden="true"></span>

Formulários

A ideia dos formulários do ANIE foi seguir tendências e diferenciar um pouco os inputs da "cara do Bootstrap".

Formulário .form-anie

<form role="form">
  <div class="form-group">
    <input type="email" class="form-anie bord-green-3" placeholder="Email address">
  </div>
  <div class="form-group">
    <input type="password" class="form-anie bord-green-3" placeholder="Password">
  </div>
  <p class="text-right">
    <button type="submit" class="btn btn-anie-2 btn-green-3">Enviar Formulário</button>
  </p>
</form>


Formulário .form-anie-2

<div class="padding black">
  <form role="form">
    <div class="form-group">
      <input type="email" class="form-anie-2" placeholder="Email address">
    </div>
    <div class="form-group">
      <input type="password" class="form-anie-2" placeholder="Password">
    </div>
    <p class="text-right"><button type="submit" class="btn transparent text-white">
      Enviar Formulário
    </button></p>
  </form>
</div>
